1. Create databases (habits and habits_test):

```psql < habits.sql```

2. Run a server:

``` cd habits-backend```
``` node server.js```

    The server will start on http://localhost:3007

3. Change a working directory and start an app:

``` cd habits-frontend```
``` npm start ```

The application will be open in a browser.
